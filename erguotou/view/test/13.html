多语言

注册多语言插件

app := erguotou.New()
app.Use(erguotou.Local("en"))

注册后第一次运行会生成多语言JSON local/local.json

{
  "zh" : {
    "name": "名称"
  },
  "fr" : {
    "name": "fr name"
  }
}
